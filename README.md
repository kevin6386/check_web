# check_web
window 下自定义url列表，自定义ip列表进行自动打开浏览器进行访问url

配置文件说明


#定义域名,前边要加'-'
Domain_name=-interface.test.com

#定义ip文件前边要加'-'

ip_file=-init.txt

#定义url前边要加'-'

Url=-url.txt

#定义时间，毫秒前边要加'-'

Time=-10000

#定义系统使用的默认浏览器

browser=-firefox.exe



#备份host文件，以管理员身份运行

#init.txt 

填写ip地址

#url.txt

填写url地址路径即可
